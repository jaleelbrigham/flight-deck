✈️ flight deck 🌴

flight deck is a data project focused on understanding how airline delays behave across a connected system.

instead of treating delays as isolated events, this project looks at flights as part of a network where disruptions can spread and impact everything downstream.

the goal is to explore how delays form, how they propagate, and what that says about the stability of the air travel system.

this project is being built step-by-step with a focus on:
	•	understanding the data first
	•	identifying patterns that actually matter
	•	slowly building toward modeling system behavior

📌 status

work in progress — currently focused on building a strong foundation for analyzing delay behavior
